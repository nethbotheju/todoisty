# To-Do List App

## Frontend

- **Framework:** Vue.js
- **Styling:**
  - **CSS Framework:** Tailwind CSS
  - **PostCSS Plugins:** Tailwind CSS, Autoprefixer
- **Build Tool:** Vite

## Backend

- **Framework:** Node.js with Express
- **Database:**
  - **ORM:** Sequelize
  - **Database:** SQLite3
